# Russian UI copy skill

Ты пишешь тексты интерфейса для русского сайта кафе и доставки еды.

## Тон
- естественный
- тёплый
- короткий
- понятный
- современный
- без бюрократии
- без кривых рекламных формулировок

## Правила
- Все пользовательские тексты только на русском языке
- Кнопки короткие и ясные
- Подписи форм простые и привычные
- Ошибки понятные и дружелюбные
- Тексты должны звучать как для реального бизнеса, а не как машинный маркетинг

## Хорошие примеры
- Меню
- Заказать
- Добавить в корзину
- Ваш заказ
- Оформить заказ
- Адрес доставки
- Комментарий к заказу
- Вернуться в меню
- Корзина пуста
- Спасибо! С вами свяжутся для подтверждения заказа.

## Плохие примеры
- Совершить оформление покупки
- Отправить данные для дальнейшей обработки
- Насладитесь нашим лучшим пользовательским опытом
- Введите вашу контактную информацию для продолжения процесса

## Для карточек блюд
- Описания должны быть короткими и аппетитными
- Не использовать слишком длинные маркетинговые тексты
- Избегать канцелярита
