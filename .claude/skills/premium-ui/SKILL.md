# Premium UI skill

Ты создаёшь современный премиальный интерфейс для сайта кафе и доставки еды.

## Главная цель
Сделать визуально сильный, тёплый, современный и аккуратный интерфейс, который не выглядит как типовой AI-шаблон.

## Визуальные принципы
- premium, warm, editorial
- вкусная и аппетитная подача
- хороший ритм секций
- сильная типографическая иерархия
- аккуратные карточки
- продуманные hover/focus состояния
- чистый mobile-first интерфейс
- качественная композиция блоков
- хорошее использование свободного пространства
- единый стиль элементов

## Нельзя делать
- generic SaaS hero section
- случайные градиенты
- однотипные блоки без визуального характера
- слишком большие скругления у всего подряд
- скучные карточки меню
- дефолтный shadcn/ui без адаптации под бренд
- визуально пустой интерфейс
- визуально шумный интерфейс

## Как работать
Перед кодом:
1. кратко описать концепцию страницы
2. описать визуальное направление
3. описать иерархию контента
4. описать разницу между mobile и desktop

Во время реализации:
- делать блоки выразительными, но чистыми
- делать кастомную композицию секций
- следить за качеством типографики
- не перегружать страницу
- сохранять ощущение реального премиального бренда

Перед завершением:
1. проверить, не выглядит ли страница шаблонной
2. проверить баланс между текстом и визуалом
3. проверить кнопки и CTA
4. проверить отступы
5. проверить мобильную версию
6. если дизайн generic — улучшить его до более кастомного уровня

## Дополнительные ограничения
- Не превращать сайт в дизайнерский концепт без коммерческой пользы
- Важнее удобство заказа, чем декоративные эффекты
- Любой красивый приём должен помогать восприятию, а не мешать ему
- Если hero section выглядит эффектно, но не помогает быстро понять меню и заказ, его нужно упростить
- Визуальная оригинальность не должна ухудшать скорость восприятия страницы
