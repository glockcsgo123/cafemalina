# Ecommerce checkout skill

Ты реализуешь корзину и оформление заказа для сайта кафе и доставки еды.

## Что обязательно должно быть
- Корзина с добавлением товаров
- Изменение количества
- Удаление товаров
- Подсчёт суммы
- Понятное пустое состояние корзины
- Простое оформление заказа
- Русские подписи полей
- Удобство на мобильных устройствах

## Поля оформления заказа
Использовать такие поля:
- Имя
- Телефон
- Адрес доставки
- Подъезд / этаж / квартира
- Комментарий к заказу

## Валидация
- Имя обязательно
- Телефон обязателен
- Адрес обязателен
- Сообщения об ошибках должны быть понятными и короткими
- Не перегружать форму лишними шагами

## Финальное состояние
После отправки заказа показать экран или блок с сообщением:

"Спасибо! С вами свяжутся для подтверждения заказа."

## UX-правила
- Checkout должен быть коротким
- Не превращать оформление в длинную анкету
- Итоговая сумма должна быть хорошо заметна
- Кнопка оформления должна быть понятной
- Интерфейс должен вызывать доверие
- Не имитировать онлайн-оплату, если её нет

## Предпочтение по реализации
- Для MVP допустим локальный state корзины
- Код должен быть понятным и чистым
- Все тексты интерфейса должны быть на русском
